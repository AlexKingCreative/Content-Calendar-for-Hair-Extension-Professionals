# Hair Pro Mobile App (Expo)

This is the Expo React Native version of the Hair Pro Content Calendar app.

## Setup

1. Navigate to the mobile directory:
   ```
   cd mobile
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the Expo development server:
   ```
   npm start
   ```

4. Scan the QR code with Expo Go app on your phone

## Features

- Email/password authentication
- Today's post view with copy and share
- Monthly calendar navigation
- Settings with Stripe subscription upgrade
- Warm rose gold theme matching the web app

## Tech Stack

- Expo SDK 52
- React Navigation
- TanStack React Query
- Expo Secure Store for token storage
- Expo Web Browser for Stripe checkout
